Lab Objective

This session introduces frequent word pattern discovery using cleaned YouTube datasets. Instead of formal theory alone, we blend manual inspection with algorithmic mining (Apriori) to find meaningful token co-occurrences. The goal: Transform raw tokens into contextual insights.

A. The Core Concept: Co-occurrence

When two or more words appear together across multiple rows of data, they "co-occur." This connection reveals context without reading every single comment manually.

D. Algorithmic: The Apriori Method

Apriori detects itemsets that meet a minimum threshold. It generates rules: "If a comment contains X, it likely contains Y."

Support

How popular is the itemset?

Freq(A,B) / Total
Confidence

How likely is Y given X?

Freq(A,B) / Freq(A)
Lift

Is the link stronger than random chance?

Conf(A->B) / Supp(B)
Apriori Workflow
  1. One-hot encode transaction list (using mlxtend).
  2. Run Apriori with min_support (e.g., 0.3, 0.2).
  3. Generate association rules & Calculate metrics.
  4. Filter: Confidence >= 0.6, Lift >= 1.2.

Exploration & Insights

E. Exploration Using Variations

Algorithms are sensitive. Tweaking one number (like minimum support) can flood you with patterns or return nothing. Key experiment: Compare rules generated from merged datasets (Comments + Captions) versus separate ones.

Tuning Tasks:
  • Lower min_support (0.15, 0.05).
  • Swap lemmatization for stemming.
  • Remove tokens under 4 characters.

F. Interpretation & Visualization
