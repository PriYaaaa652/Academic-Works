## 📖 Course Overview

This course introduces the core concepts of data communication and computer networks, including communication models, protocols, transmission techniques, and network architectures such as OSI and TCP/IP.



## 🎯 Learning Focus

* Data communication concepts and architecture
* Transmission techniques and performance analysis
* Error detection and correction methods
* Practical networking and simulation skills



## 🧪 Lab Work

* Lab Experiments & Reports
* Lab Manuals
* Mini Project

### Topics Covered

* Network devices and topology
* Router & switch configuration
* Wired and wireless LAN setup
* IP addressing and subnetting
* Packet Tracer simulations
* TCP/IP & OSI model analysis
