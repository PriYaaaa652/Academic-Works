# CSE477 Data Mining Labs (Spring 2026)

This repository contains my lab experiments and implementations for the course **CSE477: Data Mining**. The work focuses on practical text mining and analysis using YouTube comments and captions collected from a real educational video.

## Dataset Source

YouTube video used for all labs:

🔗 https://youtu.be/h3M00JI8Iwo?si=l3LrMUidwSr-CLVL

Data collected from the video:
- YouTube comments
- English auto-generated captions/subtitles

The raw data was processed and converted into:
- `cleaned_comments.csv`
- `cleaned_captions.csv`

These cleaned datasets were later reused across multiple lab experiments.

## Lab Areas Covered

- Exploratory Data Analysis (EDA)
- Text Cleaning & Preprocessing
- Tokenization & Lemmatization
- Frequent Pattern Mining
- Association Rule Mining
- Incremental Pattern Analysis
- Correlation Analysis
- TF-IDF Keyword Extraction
- K-Means Clustering
- DBSCAN Clustering
- Sentiment Analysis
- N-gram & Co-occurrence Analysis
